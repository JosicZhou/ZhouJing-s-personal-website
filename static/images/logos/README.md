# Logo Images Directory

请将以下AI产品的logo图片放到这个目录中：

## 需要的Logo文件

1. **MusicCreator AI Logo**
   - 文件名：`musiccreator-ai-logo.svg` 或 `musiccreator-ai-logo.png`
   - 建议尺寸：120x60px (最大)
   - 来源：https://www.musiccreator.ai/

2. **Coloringbook AI Logo**
   - 文件名：`coloringbook-ai-logo.svg` 或 `coloringbook-ai-logo.png`
   - 建议尺寸：120x60px (最大)
   - 来源：https://www.coloringbook.ai/

3. **Inker.AI Logo**
   - 文件名：`inker-ai-logo.svg` 或 `inker-ai-logo.png`
   - 建议尺寸：120x60px (最大)
   - 来源：https://www.inker.ai/

4. **Web Picture Scraper Logo**
   - 文件名：`web-scraper-logo.svg` 或 `web-scraper-logo.png`
   - 建议尺寸：120x60px (最大)
   - 说明：可以创建一个简单的图标，表示网页爬虫功能

5. **NanoBanana AI Logo**
   - 文件名：`nanobanana-ai-logo.svg` 或 `nanobanana-ai-logo.png`
   - 建议尺寸：120x60px (最大)
   - 来源：https://nanobanana.art/

## 注意事项

- 推荐使用SVG格式，它可以在任何尺寸下保持清晰
- 如果使用PNG格式，请确保是高分辨率的
- Logo应该在白色背景上看起来清晰
- 所有logo会自动添加阴影效果

## 如果没有logo图片

如果暂时没有某个产品的logo，可以：
1. 访问对应的网站获取logo
2. 或者创建一个简单的文字logo
3. 或者使用占位符图片

代码会自动处理图片加载失败的情况。
