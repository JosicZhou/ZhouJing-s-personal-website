# 头像文件夹

这个文件夹用于存放聊天界面的头像图片。

## 文件说明：

1. **user-avatar.jpg** - 用户头像（周晶的个人照片）
   - 建议尺寸：至少 128x128 像素
   - 格式：JPG, PNG 或 WebP
   - 比例：正方形（1:1）
   - 注意：图片会自动裁剪为圆形显示

## 如何添加头像：

1. 将你的个人照片重命名为 `user-avatar.jpg`
2. 将文件放入此文件夹：`/static/images/avatars/`
3. 刷新网页即可看到效果

## 备用方案：

如果没有上传个人照片，系统会自动使用默认的用户图标。

## 支持的文件格式：

- user-avatar.jpg
- user-avatar.png
- user-avatar.webp

优先级：jpg > png > webp
